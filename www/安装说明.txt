1. 安装环境需求
IIS/Apache + PHP5 + MySQL5 + JDK7

2. 初次安装程序，将程序上传至web服务器，根据安装界面的示完成系统的安装。

3. 安装成功后可删除 install 目录， 后台管理的文件为 m.php, 如果担心后台地址的泄漏，可直接将该文件改名，同时访问改名后的文件

4. 后台默认管理员的帐户为: admin 密码: admin
